# Snake Game (C++ + raylib)

A simple **snake game** written in **C++** using the **raylib** graphics library.

![Game Demo](media/snake_game.png)

## Requirements

Make sure you have the following installed:

- `g++`
- `make`
- `cmake`

## Build

Compile the project:

```bash
make
```

---

## Run

```bash
make run
```

---

## Clean Build Files

```bash
make clean
```
